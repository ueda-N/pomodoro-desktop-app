# 📌 最前面固定ポモドーロタイマー (Electron App)

デスクトップ専用のポモドーロタイマーアプリです。
PC起動時から常にデスクトップの片隅で作業に寄り添います。

## ✨ 主な機能

* **最前面にピン留め**
* **全自動の25分/5分無限ループ**
  * 集中（25分・黒文字）と休憩（5分・青文字）が、OS通知を飛ばしながら自動でエンドレスに切り替わります。
* **タスク名登録・記録**
  * タスク名を入力して作業時間を記録することが可能です。
* **過去10時間のタイマーログ**
  * 過去10時間分、タスク名ごとに作業時間を記録します。（5分の休憩も作業時間に含めて計算します）
  * 25分/5分の終了時だけでなく、途中で「Stop」を押した瞬間までの時間や「1分未満」の作業も合計利用時間として自動集計します。

## 🛠 開発環境 / 使用技術

* HTML5 / CSS3
* JavaScript (Vanilla JS)
* Node.js
* Electron (v30.0.0)
* electron-builder (パッケージング)

## 🚀 使い方（開発者向け）

### 1. リポジトリのクローンと依存関係のインストール
```bash
npm install
```
### 2. アプリの起動（テスト）
```bash
npm start
```
### 3. デスクトップアプリ（.exe）として書き出し
```bash
npm run build
```
実行後、dist/ フォルダ内に独立して動く .exe ファイルが生成されます。
